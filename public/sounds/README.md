# Sons de Notificação

## Arquivo Necessário

Adicione um arquivo de som chamado `notification.mp3` nesta pasta.

### Opções:

1. **Som customizado**: Adicione seu próprio arquivo MP3
2. **Sons gratuitos**: Baixe de sites como:
   - [Mixkit](https://mixkit.co/free-sound-effects/notification/)
   - [Freesound](https://freesound.org/)
   - [Notification Sounds](https://notificationsounds.com/)

### Requisitos:

- **Formato**: MP3
- **Nome**: `notification.mp3`
- **Tamanho**: Recomendado < 100KB
- **Duração**: Recomendado < 2 segundos

### Exemplo de som recomendado:

Um som discreto, curto e agradável que não seja incômodo.
